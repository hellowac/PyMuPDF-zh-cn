Welcome to PyMuPDF#

PyMuPDF 是一个高性能 Python 库，用于提取、分析、转换和处理 PDF （和其他）文档的数据。

PyMuPDF 托管在 GitHub 上，并在 PyPI 上注册。

本文档涵盖直至 1.25.4 的所有版本。
